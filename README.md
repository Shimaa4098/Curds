
# CURDS - Simple CRUD Application with JavaScript

This is a simple CRUD (Create, Read, Update, Delete) application built with plain JavaScript.  
I followed along with the tutorial by Abdelrahman Gamal to build this project and solidify my understanding of JavaScript fundamentals and DOM manipulation.

## Features
- Add new items
- View existing items
- Edit items
- Delete items

## Technologies Used
- JavaScript
- HTML
- CSS

## How to Run
Simply open the index.html file in your web browser to use the application.
